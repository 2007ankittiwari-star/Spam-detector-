# 📩 Spam Message Classifier (AI/ML Project)

## 📌 Overview

This project is a simple machine learning application that classifies SMS messages as **Spam** or **Not Spam**.

It uses basic Natural Language Processing (NLP) techniques and a Naive Bayes model.
A simple UI is also created using Tkinter where the user can enter a message and check the result.

---

## 🎯 Objective

To build a model that can automatically detect spam messages based on their content.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Tkinter

---

## 🧠 Machine Learning Concepts Used

* Text Vectorization (CountVectorizer)
* Naive Bayes Algorithm
* Train-Test Split
* Model Accuracy

---

## 📂 Project Structure

```
Spam_Project/
│── spam_ui.py
│── spam.csv
```

---

## ▶️ How to Run the Project

1. Make sure Python is installed
2. Install required libraries:

```
pip install pandas scikit-learn
```

3. Keep `spam.csv` file in the same folder as the code
4. Run the program:

```
python spam_ui.py
```

---

## 💻 Features

* Classifies messages as Spam or Not Spam
* Simple and easy-to-use UI
* Displays model accuracy
* Fast prediction

---

## 🧪 Example

Input:
`Win a free lottery now`

Output:
`Spam ❌`

The model gives around **95–98% accuracy** depending on dataset split.
